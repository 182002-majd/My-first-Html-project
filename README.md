Majd Store - E-commerce Project

Welcome to **Majd Store**, my first professional web development project. This project is a journey of turning a concept into a fully functional web interface, focusing on clean code and best practices.

##  About the Project
Majd Store is a modern clothing e-commerce platform. Currently, it focuses on building a solid structural foundation using **HTML5**, ensuring that the user journey from browsing products to the checkout page is seamless and logical.

##  Features & Milestones
- [x] **Semantic HTML5:** Built using SEO-friendly tags for better accessibility.
- [x] **Multi-page Navigation:** Fully linked Homepage and Checkout page.
- [x] **Form Integration:** A structured checkout form to capture user data.
- [ ] **CSS Styling:** (In Progress) Adding a modern and responsive UI.
- [ ] **JavaScript Logic:** (Upcoming) Adding interactivity and cart functionality.
